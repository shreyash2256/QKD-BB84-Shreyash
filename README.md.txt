# BB84 Quantum Key Distribution Simulation

##  Project Description
This project implements a complete end-to-end simulation of the BB84 Quantum Key Distribution (QKD) protocol.

##  Features
- Raw key generation
- Random basis selection
- Quantum encoding & measurement
- Eavesdropping simulation
- Basis sifting
- Error rate calculation
- Privacy amplification using SHA-256

## Technologies Used
- Python

##  How to Run
1. Open terminal in project folder
2. Run:
   python main.py

##  Output
- Alice and Bob keys
- Error rate
- Final secure key

## Files
- main.py → Main simulation code
- README.md → Project description
- report.pdf → Documentation
